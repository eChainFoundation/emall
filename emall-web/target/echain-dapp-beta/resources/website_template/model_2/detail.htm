触手可及的异国美景  海南梦想旅行地

2014年1月7日

     落地海口，直奔骑楼。海口骑楼老街和泰国普吉镇老城区的建筑都是南洋风格，墙上的斑驳感证明了它们拥有同样的悠久历史，而新建的部分又都是模仿老建筑的风格而更加相似。除了外表上的接近，骑楼和普集镇老城都是当地人采买生活用品和吃夜宵的地方，特别能体现本土的生活氛围。海口骑楼老街中最古老的建筑四牌楼建于南宋，至今有700多年历史。骑楼大多是上世纪初一批批从南洋回来的华侨借鉴当时的南洋建筑风格所建。骑楼外表乍看斑驳古拙，仔细欣赏时发现那些建筑上大多布满优雅细致的雕塑和洋派的装饰，很有些巴洛克的味道。
